# Gestor de Productos Python

Aplicación de escritorio desarrollada en Python para la gestión de productos utilizando una base de datos SQLite.

## Descripción

Este proyecto permite registrar, consultar, modificar y eliminar productos mediante una interfaz gráfica desarrollada con Tkinter. Los datos se almacenan de forma permanente en una base de datos SQLite.

## Funcionalidades

- Añadir productos.
- Consultar productos almacenados.
- Modificar información de productos.
- Eliminar productos.
- Almacenamiento persistente mediante SQLite.
- Interfaz gráfica sencilla e intuitiva.

## Tecnologías utilizadas

- Python
- Tkinter
- SQLite
